## 🌐 웹 세션 - 사전 준비사항 상세 가이드

웹 세션에서는 Gemini를 활용한 웹 개발부터 실제 사이트 배포까지, 웹 개발의 전체적인 흐름을 경험하게 됩니다. 이를 위해 여러분의 컴퓨터에 웹 개발에 필요한 전문적인 도구들을 설치하고, 관련 서비스에 가입하는 과정이 필요합니다. 아래 가이드를 차근차근 따라와 주세요.

### 1\. 필수 계정 생성하기

웹사이트를 만들고, 디자인하며, 전 세계에 공개(배포)하기 위해 아래 계정들이 필요합니다. 모두 무료로 만들 수 있습니다.

#### **구글 계정 (Google Account)**

  * **왜 필요한가요?** 이번 세션에서 우리가 만들 웹사이트를 무료로 배포하고 운영할 수 있게 해주는 **Firebase** 서비스를 사용하기 위해 필요합니다. Firebase는 구글이 제공하는 서비스입니다.
  * **가입 방법:** (AI 세션에서 이미 만드셨다면 이 단계는 건너뛰세요.)
    1.  웹 브라우저에서 [**google.com**](https://www.google.com)으로 접속한 뒤, 오른쪽 위의 **'로그인'** 버튼을 누르세요.
    2.  로그인 화면에서 \*\*'계정 만들기'\*\*를 클릭하고, \*\*'내 개인용도'\*\*를 선택하세요.
    3.  화면에 나오는 안내에 따라 이름, 생년월일, 성별 등을 입력하고 원하는 Gmail 주소와 비밀번호를 만드세요.

#### **Builder.io 계정 (Builder.io Account)**

  * **왜 필요한가요?** 코드를 잘 몰라도 마우스로 끌어다 놓는(Drag-and-Drop) 방식으로 웹페이지 UI를 디자인하고, 그 결과를 실제 코드로 변환해주는 '로우코드(Low-code)' 도구입니다. 이번 세션에서 디자인 생산성을 높이는 경험을 위해 사용합니다.
  * **가입 방법:**
    1.  웹 브라우저에서 [**builder.io**](https://www.builder.io/)로 접속하세요.
    2.  오른쪽 위 `Sign Up` 또는 중앙의 `Get started for free` 버튼을 클릭하세요.
    3.  **`Continue with Google` 또는 `Continue with GitHub`를 클릭**하여 방금 만든 구글 또는 GitHub 계정으로 간편하게 가입하는 것을 추천합니다.

### 2\. 필수 소프트웨어 설치하기

이제 여러분의 컴퓨터를 실제 웹 개발이 가능한 환경으로 만들어 보겠습니다.

#### **구글 크롬 브라우저 (Google Chrome)**

  * **왜 필요한가요?** 뛰어난 개발자 도구를 내장하고 있어 웹 개발에 가장 널리 사용되는 브라우저입니다.
  * **설치 방법:** (이미 설치되어 있다면 건너뛰세요.)
    1.  [**google.com/chrome**](https://www.google.com/chrome/) 주소로 접속하세요.
    2.  **'Chrome 다운로드'** 버튼을 클릭하여 설치 파일을 받은 뒤 실행하여 설치를 완료합니다.

#### **Visual Studio Code (VS Code)**

  * **왜 필요한가요?** 전 세계 개발자들이 가장 사랑하는 무료 코드 에디터(메모장)입니다. 코드를 작성하고, 수정하고, 관리하는 모든 작업을 여기서 하게 됩니다.
  * **설치 방법:**
    1.  [**code.visualstudio.com**](https://code.visualstudio.com/) 주소로 접속하세요.

    2.  메인 화면에 내 컴퓨터 운영체제(Windows/Mac)에 맞는 **`Download`** 버튼이 보입니다. 클릭하여 설치 파일을 다운로드하세요.

    3.  다운로드된 파일을 실행하고, 안내에 따라 설치를 진행하세요. 특별히 변경할 옵션 없이 계속 \*\*'다음' 또는 '계속'\*\*을 눌러 설치를 완료하면 됩니다.

#### **Node.js 및 npm**

  * **왜 필요한가요?** 웹사이트 배포 도구(`Firebase CLI`)를 비롯한 다양한 웹 개발 도구들을 설치하고 사용하기 위해 반드시 필요한 기본 실행 환경입니다. `npm`은 Node.js를 설치하면 자동으로 함께 설치되는 '앱 스토어' 같은 프로그램입니다.
  * **설치 방법:**
    1.  [**nodejs.org**](https://nodejs.org/ko/) 주소로 접속하세요.

    2.  두 개의 버전 중 **LTS** (Long Term Support)라고 적힌 버전을 클릭하여 다운로드하세요. LTS 버전이 가장 안정적이고 호환성이 좋습니다.

    3.  다운로드된 파일을 실행하고, VS Code와 마찬가지로 모든 옵션을 그대로 두고 \*\*'Next' 또는 'Continue'\*\*를 눌러 설치를 완료하세요.

### 3\. 추가 설치: Firebase CLI

CLI(Command Line Interface)는 마우스가 아닌 명령어를 입력해서 컴퓨터와 소통하는 방식입니다. 이제 이 CLI를 통해 Firebase 배포 도구를 설치해 보겠습니다.

  * **왜 필요한가요?** 터미널에서 간단한 명령어(`firebase deploy`)를 입력하는 것만으로 우리가 만든 웹사이트를 전 세계 인터넷에 공개(배포)할 수 있게 해주는 마법 같은 도구입니다.

  * **설치 방법:**

    1.  **터미널(Terminal) 실행하기:**
          * **Windows:** `시작 메뉴`를 누르고 `cmd` 또는 `명령 프롬프트`를 검색하여 실행하세요.
          * **Mac:** `Launchpad`나 `Spotlight 검색(⌘+Space)`에서 `Terminal` 또는 `터미널`을 검색하여 실행하세요.
    2.  **설치 명령어 입력하기:**
          * 검은색 터미널 창이 열리면, 아래 명령어를 그대로 복사해서 붙여넣고 `Enter` 키를 누르세요.
        <!-- end list -->
        ```bash
        npm install -g firebase-tools
        ```
    3.  잠시 기다리면 설치가 진행되고, 아무런 에러 메시지 없이 다음 입력 줄이 나타나면 성공적으로 설치된 것입니다.
